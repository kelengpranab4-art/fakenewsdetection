# TruthSense AI - Fake News Detector

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fkelengpranab4-art%2Ffakenewsdetection)

TruthSense AI is a machine-learning-powered web application that classifies news articles as **REAL** or **FAKE** with high precision.

## ✨ Features
- **High Accuracy**: 92.27% accuracy using Passive Aggressive Classifier.
- **Glassmorphic UI**: Modern, futuristic design with neon accents.
- **Real-time Prediction**: Deep analysis of news content and headlines.
- **Confidence Scoring**: Dynamic confidence percentage for each result.

## 🚀 Quick Start
1. Clone the repo: `git clone https://github.com/kelengpranab4-art/fakenewsdetection.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `python3 app.py`

## 🌍 Deployment
This repository is optimized for **Vercel** serverless deployment. Simply connect this repository to your Vercel account or click the "Deploy" button above.

## 🛠 Tech Stack
- **Backend**: Flask (Python)
- **Machine Learning**: Scikit-learn, Passive Aggressive Classifier, TF-IDF Vectorizer
- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6)
